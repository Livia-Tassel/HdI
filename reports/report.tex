\begin{abstract}
本研究基于2026年（第19届）中国大学生计算机设计大赛大数据主题赛"健康数据洞察"赛题提供的5类数据集，对2000--2023年全球203个国家和地区的疾病谱演变、风险因素归因与卫生资源配置效率开展了系统性的数据驱动分析，并对中国近20年省级卫生数据进行深度补充刻画。

研究围绕赛题要求的三大核心维度展开：\textbf{维度一}（时空变迁）构建全球国家--年份主面板，量化疾病谱从传染病向非传染性疾病（NCD）转型的长期趋势，并以截面OLS回归识别预期寿命的关键解释变量（$R^2 = 0.906$）；\textbf{维度二}（风险归因）基于归因死亡数据建立风险贡献占比框架，结合人口归因分值（PAF）与Shapley值分解量化各风险因素的边际贡献，并构建三种未来干预情景；\textbf{维度三}（资源配置）采用标准化缺口指数与投入--产出四象限分类评估各国资源匹配程度，进而以对数生产函数为基础建立约束优化模型，分别求解总产出最大化与Rawlsian最小值最大化两类再分配方案。

在赛题主线之外，本项目提出两个原创分析维度：（1）\textbf{健康公平性分析}，计算全球预期寿命的Gini系数、Theil指数与$\sigma$-收敛指标，揭示国际健康不平等的长期演化趋势；（2）\textbf{中国省级深度聚焦}，利用赛题第五类数据对31个省级行政区卫生人员与卫生机构的时序演化与区域差异进行可视化分析。研究最终形成包含可复现数据处理程序、静态API输出、8幅分析图表、6张数据表格、交互式仪表盘与完整AI协作记录的全套交付物。

\noindent\textbf{关键词：}疾病谱转型；人口归因分值；Shapley值分解；线性规划；DEA效率评估；健康公平；Gini系数；面板回归
\end{abstract}

\subsection{选题背景}

"健康中国2030"战略纲要将"改善健康结果、降低风险暴露、优化资源配置"列为核心战略目标，联合国可持续发展目标（SDG 3）同样以"确保各年龄段所有人的健康生活和促进福祉"为追求。与传统只讨论单一疾病或单一指标的做法不同，本赛题强调用多源数据描述\textbf{健康生态系统}的多维结构，并在此基础上给出可复现、可解释、可执行的洞察。

全球健康格局正经历深刻转型：传染性疾病的控制取得了显著成就，但非传染性疾病（NCD）的负担持续上升；不同地区的主导风险因素差异显著；卫生资源在国际间的分配极不均衡。这些现实要求研究者从时空维度、因果维度和决策维度三个层面同时切入，才能形成完整的分析图景。

\subsection{赛题任务与分析维度}

本项目围绕赛题要求的三个核心维度展开，并提出两个原创扩展维度：

\begin{enumerate}[label=\textbf{D\arabic*.},leftmargin=2.5em]
    \item \textbf{疾病谱时空变迁}：描述全球疾病谱从传染性疾病向NCD转移的时空过程，量化健康指标差异，并对代表国家进行趋势预测。
    \item \textbf{风险因素归因与干预优先级}：分析主要风险因素对全因死亡的归因贡献，构建未来情景模拟与分地区优先干预清单。
    \item \textbf{卫生资源配置与优化}：评估各国卫生资源投入与健康产出的匹配程度，识别资源不足或低效率区域，并给出约束优化下的再分配方案。
    \item \textbf{健康公平性分析}（原创维度）：利用不平等测度指标分析全球健康差距的长期趋势。
    \item \textbf{中国省级深度聚焦}（原创维度）：对中国31个省级行政区卫生资源的时序演化与区域差异进行深度分析。
\end{enumerate}

\subsubsection{缺失值处理策略}

对于面板数据中的缺失值，采用以下分层策略：
\begin{enumerate}[label=\arabic*)]
    \item 对缺失率超过30\%的变量列，从当次分析中排除；
    \item 对缺失间隔$\leq 2$年的变量，使用线性插值填补：
    \begin{equation}
        \hat{x}_{i,t} = x_{i,t_1} + \frac{t - t_1}{t_2 - t_1}(x_{i,t_2} - x_{i,t_1}), \quad t_1 < t < t_2
    \end{equation}
    其中$t_1, t_2$为已观测的相邻年份。
    \item 对尾部缺失，使用前向填充（last observation carried forward）。
\end{enumerate}

\subsubsection{标准化方法}

在需要进行跨变量比较时，采用Z-score标准化：
\begin{equation}
    z_j = \frac{x_j - \bar{x}_j}{\sigma_j}
\end{equation}
其中$\bar{x}_j$和$\sigma_j$分别为变量$j$在截面上的均值和标准差。需要方向反转的变量（如死亡率越高表示越差），取$z_j^\prime = -z_j$。

\section{方法论框架}
% ============================================================================

本节系统阐述各维度所采用的数学模型与统计方法。

\subsection{截面OLS回归模型}

对最新年份的截面数据，使用多元线性回归识别预期寿命的关键解释因素：

\begin{equation}
\label{eq:ols}
\text{LE}_i = \beta_0 + \beta_1 \ln(\text{GDP}_i) + \beta_2 \text{HealthExp}_i + \beta_3 \text{Phys}_i + \beta_4 \text{Water}_i + \beta_5 \text{Sanit}_i + \beta_6 \text{CommSh}_i + \beta_7 \text{NCDSh}_i + \varepsilon_i
\end{equation}

其中$\text{LE}_i$为国家$i$的出生时预期寿命，$\ln(\text{GDP}_i)$为人均GDP的对数变换（缓解右偏分布），$\text{CommSh}_i$和$\text{NCDSh}_i$分别为传染性疾病和非传染性疾病死亡占比。模型参数通过最小二乘法（OLS）估计：

\begin{equation}
    \hat{\bm{\beta}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}
\end{equation}

拟合优度以决定系数$R^2$衡量：
\begin{equation}
    R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2} = 1 - \frac{SS_{\text{res}}}{SS_{\text{tot}}}
\end{equation}

\subsection{面板固定效应模型与Driscoll-Kraay标准误}

对面板数据，估计双向固定效应模型：
\begin{equation}
\label{eq:panel-fe}
y_{it} = \alpha_i + \lambda_t + \mathbf{x}_{it}^\top \bm{\beta} + \varepsilon_{it}
\end{equation}

其中$\alpha_i$为国家固定效应（控制不随时间变化的国家异质性），$\lambda_t$为时间固定效应（控制全球共同冲击），$\mathbf{x}_{it}$为时变解释变量向量。

为处理面板截面相关性（cross-sectional dependence）与异方差问题，采用Driscoll-Kraay标准误。设$N$为截面个体数，$T$为时间跨度，$K$为自变量个数，Driscoll-Kraay协方差矩阵估计为：

\begin{equation}
    \widehat{\text{Var}}(\hat{\bm{\beta}}) = (\mathbf{X}^\top \mathbf{X})^{-1} \hat{\mathbf{S}} (\mathbf{X}^\top \mathbf{X})^{-1}
\end{equation}

其中$\hat{\mathbf{S}}$为基于Newey-West型核加权的HAC（Heteroskedasticity and Autocorrelation Consistent）估计量：

\begin{equation}
    \hat{\mathbf{S}} = \sum_{\ell = -m}^{m} w(\ell, m) \hat{\mathbf{\Gamma}}_\ell, \quad w(\ell, m) = 1 - \frac{|\ell|}{m+1}
\end{equation}

此处$\hat{\mathbf{\Gamma}}_\ell = \frac{1}{T}\sum_{t} \mathbf{h}_t \mathbf{h}_{t-\ell}^\top$为截面加总后的自协方差矩阵，$\mathbf{h}_t = \sum_{i=1}^{N} \mathbf{x}_{it}\hat{\varepsilon}_{it}$。

\subsection{Mundlak相关随机效应模型}

为检验固定效应假设的必要性，补充估计Mundlak（1978）相关随机效应（CRE）模型：

\begin{equation}
    y_{it} = \mathbf{x}_{it}^\top \bm{\beta} + \bar{\mathbf{x}}_i^\top \bm{\gamma} + \alpha_i + \varepsilon_{it}
\end{equation}

其中$\bar{\mathbf{x}}_i = \frac{1}{T_i}\sum_{t} \mathbf{x}_{it}$为个体在时间维度上的均值向量。若$\bm{\gamma}$联合显著，则意味着随机效应模型不一致，应使用固定效应。

\subsection{人口归因分值（PAF）}

人口归因分值（Population Attributable Fraction, PAF）是量化单一风险因素对疾病负担贡献的核心指标。对于二值暴露：

\begin{equation}
\label{eq:paf}
\text{PAF} = \frac{p \cdot (RR - 1)}{1 + p \cdot (RR - 1)}
\end{equation}

其中$p$为人群中该风险因素的暴露率（prevalence），$RR$为相对危险度（relative risk）。

对于多水平暴露，推广为：
\begin{equation}
    \text{PAF} = \frac{\sum_{k=1}^{K} p_k (RR_k - 1)}{1 + \sum_{k=1}^{K} p_k (RR_k - 1)}
\end{equation}

可归因死亡数为：
\begin{equation}
    D_{\text{attr}} = \text{PAF} \times D_{\text{total}}
\end{equation}

\subsection{联合PAF与独立性假设}

当存在$K$个相互独立的风险因素时，联合PAF为：
\begin{equation}
\label{eq:joint-paf}
    \text{JointPAF} = 1 - \prod_{k=1}^{K}(1 - \text{PAF}_k)
\end{equation}

此公式基于风险因素之间的统计独立性假设。当风险因素之间存在正相关时（如高BMI与高血糖），上式会低估联合归因比例，此时需要采用Shapley值分解进行修正。

\subsection{Shapley值分解}

Shapley值源于合作博弈论，用于在$n$个"玩家"（风险因素）之间公平分配"收益"（总归因疾病负担）。对风险因素$i$，其Shapley值定义为：

\begin{equation}
\label{eq:shapley}
\phi_i = \sum_{S \subseteq N \setminus \{i\}} \frac{|S|!\,(n - |S| - 1)!}{n!}\bigl[v(S \cup \{i\}) - v(S)\bigr]
\end{equation}

其中$N = \{1, 2, \ldots, n\}$为全部风险因素集合，$S$为不含$i$的任意子集，$v(S)$为联盟$S$的价值函数。在本研究中，价值函数取为联合PAF：

\begin{equation}
    v(S) = \text{JointPAF}(S) = 1 - \prod_{k \in S}(1 - \text{PAF}_k)
\end{equation}

Shapley值满足以下核心性质：
\begin{itemize}[leftmargin=1.5em]
    \item \textbf{有效性}（Efficiency）：$\sum_{i=1}^{n} \phi_i = v(N)$
    \item \textbf{对称性}（Symmetry）：等价玩家获得相同分配
    \item \textbf{虚玩家性}（Null Player）：不产生边际贡献的玩家分配为零
    \item \textbf{可加性}（Additivity）：分配相对于价值函数线性
\end{itemize}

由于精确Shapley计算的复杂度为$O(2^n)$，当风险因素数$n$较大时，可采用蒙特卡洛近似或随机排列采样。

\subsection{线性趋势预测与置信区间}

对时间序列数据，采用线性趋势外推作为基线预测模型：
\begin{equation}
\label{eq:forecast}
    \hat{y}_t = \hat{\beta}_0 + \hat{\beta}_1 \cdot t
\end{equation}

其中$\hat{\beta}_0, \hat{\beta}_1$由最小二乘法从历史数据$\{(t_k, y_k)\}_{k=1}^{n}$中估计。预测的95\%置信区间为：

\begin{equation}
    \text{CI}_{0.95}(t) = \hat{y}_t \pm 1.96 \cdot \text{RMSE}
\end{equation}

其中均方根误差为：
\begin{equation}
    \text{RMSE} = \sqrt{\frac{1}{n}\sum_{k=1}^{n}(y_k - \hat{y}_k)^2}
\end{equation}

同时报告平均绝对误差（MAE）和平均绝对百分比误差（MAPE）：
\begin{align}
    \text{MAE} &= \frac{1}{n}\sum_{k=1}^{n}|y_k - \hat{y}_k| \\
    \text{MAPE} &= \frac{100\%}{n}\sum_{k=1}^{n}\left|\frac{y_k - \hat{y}_k}{y_k}\right|
\end{align}

\subsection{资源缺口指数}

Dimension 3的资源缺口评估基于投入指数与需求指数的差值：

\textbf{投入指数}（实际资源水平）：
\begin{equation}
    I_i^{\text{input}} = \frac{1}{4}\left[z(\text{Phys}_i) + z(\text{Beds}_i) + z(\text{HExp\%}_i) + z(\text{HExpPC}_i)\right]
\end{equation}

\textbf{产出指数}（健康结果）：
\begin{equation}
    I_i^{\text{output}} = \frac{1}{3}\left[z(\text{LE}_i) + z^{-}(\text{IMR}_i) + z^{-}(\text{U5MR}_i)\right]
\end{equation}

其中$z^{-}$表示取反后的标准化值（死亡率越高越差）。

\textbf{理论需求指数}：
\begin{equation}
    I_i^{\text{need}} = \frac{1}{4}\left[z(\text{CommSh}_i) + z(\text{IMR}_i) + z(\text{U5MR}_i) + z^{-}(\text{LE}_i)\right]
\end{equation}

\textbf{资源缺口}定义为实际投入与理论需求的差值：
\begin{equation}
    \text{Gap}_i = I_i^{\text{input}} - I_i^{\text{need}}
\end{equation}

$\text{Gap}_i < 0$表示资源不足于需求；$\text{Gap}_i > 0$表示资源充裕。按五分位数将缺口分为A（富余）、B（较充足）、C（匹配）、D（不足）、E（严重不足）五级。

\subsection{投入--产出四象限分类}

以投入指数中位数$M^{\text{input}}$和产出指数中位数$M^{\text{output}}$为阈值，将各国分入四个象限：

\begin{equation}
    Q_i = \begin{cases}
    \text{Q1: 高投入--高产出} & \text{if } I_i^{\text{input}} \geq M^{\text{input}} \land I_i^{\text{output}} \geq M^{\text{output}} \\
    \text{Q2: 低投入--高产出} & \text{if } I_i^{\text{input}} < M^{\text{input}} \land I_i^{\text{output}} \geq M^{\text{output}} \\
    \text{Q3: 高投入--低产出} & \text{if } I_i^{\text{input}} \geq M^{\text{input}} \land I_i^{\text{output}} < M^{\text{output}} \\
    \text{Q4: 低投入--低产出} & \text{if } I_i^{\text{input}} < M^{\text{input}} \land I_i^{\text{output}} < M^{\text{output}} \\
    \end{cases}
\end{equation}

其中Q2象限代表在有限资源条件下仍保持较好健康结果的国家，值得作为效率经验推广对象。

\subsection{数据包络分析（DEA）}

DEA是一种非参数的效率评估方法。对于包含$n$个决策单元（DMU）的系统，每个DMU消耗$m$种投入$\mathbf{x}_j \in \mathbb{R}^m$产生$s$种产出$\mathbf{y}_j \in \mathbb{R}^s$，DMU $k$的输入导向可变规模报酬（VRS）DEA模型为：

\begin{equation}
\label{eq:dea}
\begin{aligned}
\min_{\theta, \bm{\lambda}} \quad & \theta \\
\text{s.t.} \quad & \sum_{j=1}^{n} \lambda_j y_{rj} \geq y_{rk}, \quad r = 1, \ldots, s \\
& \sum_{j=1}^{n} \lambda_j x_{ij} \leq \theta \cdot x_{ik}, \quad i = 1, \ldots, m \\
& \sum_{j=1}^{n} \lambda_j = 1 \\
& \lambda_j \geq 0, \quad j = 1, \ldots, n
\end{aligned}
\end{equation}

其中$\theta^* \in (0, 1]$为效率得分，$\theta^* = 1$表示该DMU位于生产前沿面上。约束$\sum \lambda_j = 1$体现可变规模报酬假设（VRS），若去掉则为固定规模报酬（CRS）模型。

\subsection{对数生产函数与约束优化}

\subsubsection{生产函数拟合}

以各国人均卫生支出$x_i$为投入，产出指数$y_i$为产出，拟合对数生产函数：
\begin{equation}
\label{eq:production}
    f(x) = a \cdot \ln(x + 1) + b
\end{equation}

参数$(a, b)$通过最小二乘法从截面数据中估计：
\begin{equation}
    (a^*, b^*) = \arg\min_{a, b} \sum_{i=1}^{n} \left[y_i - a \cdot \ln(x_i + 1) - b\right]^2
\end{equation}

对数函数$f(x) = a\ln(x+1) + b$满足$f''(x) < 0$（凹性），体现卫生投入的边际递减效应。

\subsubsection{总产出最大化模型}

在总预算约束下，最大化各国健康产出之和：

\begin{equation}
\label{eq:max-output}
\begin{aligned}
\max_{\mathbf{x}} \quad & \sum_{i=1}^{n} f(x_i) = \sum_{i=1}^{n} \left[a \cdot \ln(x_i + 1) + b\right] \\
\text{s.t.} \quad & \sum_{i=1}^{n} x_i \leq B \\
& x_i^{\text{min}} \leq x_i \leq x_i^{\text{max}}, \quad i = 1, \ldots, n
\end{aligned}
\end{equation}

其中$B = \alpha \cdot \sum_i x_i^{\text{current}}$为预算总量（$\alpha$为预算乘数），$x_i^{\text{min}} = 0.5 \cdot x_i^{\text{current}}$和$x_i^{\text{max}} = 2.0 \cdot x_i^{\text{current}}$为每国的合理变动范围。

由于目标函数为可分离凹函数，最优解具有\textbf{注水（Water-Filling）}结构：在无限制条件下，各国应获得相等的资源配置水平；受限条件下，通过二分法求解共同水位线$\mu$：

\begin{equation}
    x_i^* = \text{clip}(\mu, \; x_i^{\text{min}}, \; x_i^{\text{max}}), \quad \text{使得} \; \sum_i x_i^* = B
\end{equation}

\subsubsection{Rawlsian最小值最大化模型}

基于Rawls正义论的"最大最小原则"（maximin），最大化最弱势国家的健康产出：

\begin{equation}
\label{eq:maximin}
\begin{aligned}
\max_{\mathbf{x}, \tau} \quad & \tau \\
\text{s.t.} \quad & f(x_i) \geq \tau, \quad i = 1, \ldots, n \\
& \sum_{i=1}^{n} x_i \leq B \\
& x_i^{\text{min}} \leq x_i \leq x_i^{\text{max}}
\end{aligned}
\end{equation}

此模型通过引入辅助变量$\tau$将max-min问题转化为线性约束下的凹规划，使用SLSQP（序列最小二乘规划）求解器进行数值求解。

\subsection{健康公平性测度指标}

\subsubsection{Gini系数}

Gini系数衡量预期寿命在国家间的分布不平等程度。设$n$个国家的预期寿命升序排列为$y_1 \leq y_2 \leq \cdots \leq y_n$，则：

\begin{equation}
\label{eq:gini}
    G = \frac{\sum_{i=1}^{n}\sum_{j=1}^{n}|y_i - y_j|}{2n^2 \bar{y}}
\end{equation}

等价的秩公式为：
\begin{equation}
    G = \frac{2\sum_{i=1}^{n} i \cdot y_i}{n \sum_{i=1}^{n} y_i} - \frac{n+1}{n}
\end{equation}

$G \in [0, 1]$，$G = 0$表示完全平等，$G = 1$表示极端不平等。

\subsubsection{Theil指数}

Theil指数（广义熵指数$\text{GE}(1)$）可分解为组间与组内不平等：

\begin{equation}
\label{eq:theil}
    T = \frac{1}{n}\sum_{i=1}^{n}\frac{y_i}{\bar{y}}\ln\frac{y_i}{\bar{y}}
\end{equation}

可分解为：
\begin{equation}
    T = T_{\text{between}} + T_{\text{within}} = \sum_{g} s_g \ln\frac{s_g}{f_g} + \sum_{g} s_g T_g
\end{equation}

其中$s_g = \sum_{i \in g} y_i / \sum_i y_i$为组$g$的收入份额，$f_g = n_g / n$为人口份额，$T_g$为组内Theil指数。在本研究中，以WHO地区为分组变量，分析健康不平等的区域间与区域内成分。

\subsubsection{$\sigma$-收敛}

$\sigma$-收敛用预期寿命对数值的截面标准差衡量：

\begin{equation}
\label{eq:sigma}
    \sigma_t = \sqrt{\frac{1}{n}\sum_{i=1}^{n}\left[\ln(y_{it}) - \overline{\ln(y_t)}\right]^2}
\end{equation}

$\sigma_t$随时间下降表示各国预期寿命出现向均值收敛的趋势。

利用公式~\eqref{eq:gini}计算2000--2023年各年的预期寿命Gini系数。结果显示：

\begin{itemize}[leftmargin=1.5em]
    \item 2000年Gini系数约为0.06--0.08，表明全球预期寿命分布的不平等程度总体上不算极端（相对于收入Gini通常在0.3--0.6的范围）。
    \item Gini系数呈缓慢下降趋势，表明全球健康不平等在\textbf{逐步缓解}。
    \item 但下降速度在2015年后有所减缓，暗示"容易摘的果子"已经接近摘完，进一步缩小差距需要更大的政策努力。
\end{itemize}

\subsection{$\sigma$-收敛分析}

$\sigma$-收敛指标（公式~\eqref{eq:sigma}）给出了更直接的离散度衡量。预期寿命对数值的截面标准差同样呈下降趋势，与Gini系数的结论一致，验证了全球健康水平的收敛趋势。

\subsection{Theil指数分解}

利用公式~\eqref{eq:theil}将健康不平等分解为WHO地区间（between-region）与地区内（within-region）两部分。结果表明：

\begin{itemize}[leftmargin=1.5em]
    \item \textbf{区域间不平等}是总不平等的主要来源，AFRO与EURO/WPRO之间的预期寿命差距是全球健康鸿沟的核心成分。
    \item \textbf{区域内不平等}在AFRO和SEARO地区内部较大，反映出这些区域内部各国发展水平的异质性。
\end{itemize}

\subsection{政策启示}

健康公平性分析的核心启示是：全球健康不平等虽在缓解，但速度递减；未来缩小差距的关键在于AFRO地区——只有显著提升非洲国家的基础卫生条件和传染病防控能力，才能实现更实质性的全球健康收敛。

\section{原创维度二：中国省级深度聚焦}

\subsection{数据与方法}

利用赛题第五类数据（中国近20年卫生数据），分析31个省级行政区在2005--2024年间卫生人员数量和卫生机构数量的时序演化与区域差异。

设省份$p$在年份$t$的卫生人员数量为$H_{p,t}$（万人），卫生机构数量为$F_{p,t}$（个），计算以下指标：

\textbf{年均增长率}：
\begin{equation}
    \text{CAGR}_p = \left(\frac{H_{p, T}}{H_{p, t_0}}\right)^{1/(T - t_0)} - 1
\end{equation}

\textbf{变异系数}（衡量省际差异）：
\begin{equation}
    \text{CV}_t = \frac{\sigma_t}{\bar{H}_t}
\end{equation}

\subsection{卫生人员与机构趋势}

图~\ref{fig:china-trends}展示了中国重点省份的卫生资源长期趋势。

\begin{figure}[H]
    \centering
    \includegraphics[width=0.9\textwidth]{figures/fig08_china_province_trends.png}
    \caption{中国重点省份2005--2024年卫生人员与机构数量趋势}
    \label{fig:china-trends}
\end{figure}

主要发现：
\begin{enumerate}[label=\arabic*)]
    \item \textbf{规模分化显著}：广东、山东、河南等人口大省在卫生人员绝对数量上遥遥领先；北京、上海虽绝对数量不突出，但人均密度优势明显。
    \item \textbf{增长趋势普遍向好}：几乎所有省份的卫生人员和机构数量在20年间均有显著增长，反映"健康中国"战略下卫生体系建设的持续投入。
    \item \textbf{西部追赶效应}：西藏、青海、宁夏等西部省份虽基数较低，但增长率相对较高，呈现一定程度的追赶趋势。

\section{结论与讨论}
% ============================================================================

\subsection{核心结论}

本研究基于赛题提供的5类数据集，完成了一个围绕"疾病谱--风险归因--资源配置"的系统性全球健康分析框架。核心结论如下：

\begin{enumerate}[label=\textbf{C\arabic*.}]
    \item \textbf{疾病谱转型已深入}：2023年全球NCD死亡占比达74.57\%，但低寿命国家仍表现出60\%以上的传染性疾病占比，说明"双重疾病负担"是低收入国家的核心挑战。截面OLS回归（$R^2 = 0.906$）确认人均GDP对数是预期寿命最显著的解释变量。

    \item \textbf{高血压是全球最重要的可干预风险因素}：2023年全球归因死亡约1083.5万人。但AFRO的首要矛盾不是慢病风险，而是儿童和孕产妇营养不良（16.2\%），揭示了不同发展阶段国家面临的差异化风险格局。

    \item \textbf{卫生资源缺口高度集中于AFRO}：前15个"严重不足"国家中14个位于非洲。同时，26个"低投入--高产出"国家提示了效率经验复制的可能性。

    \item \textbf{约束优化提供了可操作的再分配方案}：总产出最大化倾向于均等化投入水平（注水效应），maximin则向最弱势国家集中——两种方案的对比为政策制定者提供了"效率--公平"权衡的量化参考。

    \item \textbf{全球健康不平等正在缓解但速度递减}：Gini系数和$\sigma$-收敛指标均呈下降趋势，但2015年后减缓，进一步收敛需要更大的政策投入。

    \item \textbf{中国省级卫生资源快速增长但省际差距显著}：20年间各省卫生人员和机构数量普遍翻番，但东西差距仍然是卫生公平的核心挑战。
\end{enumerate}

\subsection{创新点}

\begin{enumerate}[label=\arabic*)]
    \item \textbf{多维度融合分析}：将疾病谱演变、风险归因与资源配置放入统一分析框架，避免"看病不看因、看因不看治、看治不看钱"的碎片化视角。
    \item \textbf{Optimization Lab交互模块}：将静态优化结果升级为可切换目标、可切换预算的决策实验界面，增强了分析的可交互性和政策实用性。
    \item \textbf{健康公平性原创维度}：引入Gini系数、Theil指数和$\sigma$-收敛三种不平等测度，从分配正义视角补充了主线分析。
    \item \textbf{中国省级深度聚焦}：利用赛题第五类数据对中国卫生资源进行空间异质性分析，契合"健康中国"战略背景。
    \item \textbf{全链路可复现}：从原始数据到最终图表、仪表盘的全流程由Python脚本驱动，任何人均可在相同环境中复现全部结果。
\end{enumerate}

\subsection{局限性与展望}

\begin{enumerate}[label=\arabic*)]
    \item 赛题第二类数据提供的是"风险因素归因死亡数"而非暴露率和RR参数，因此Dimension 2的主口径为"贡献占比分析"而非教科书意义上的PAF估计。本项目已实现完整的PAF和Shapley值分解代码，在有标准参数时可直接切换。
    \item 线性趋势预测未纳入非线性因素（如COVID-19冲击、政策突变等），实际轨迹会有更大不确定性。
    \item DEA效率评估对投入/产出变量的选择较为敏感，未来可引入超效率DEA（Super-efficiency DEA）和Malmquist生产率指数进行稳健性检验。
    \item 中国省级分析受限于数据可用性，未包含人均指标和健康结果指标，仅从绝对规模角度刻画资源差异。
\end{enumerate}
